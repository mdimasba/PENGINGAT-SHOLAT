> Pengingat waktu sholat langsung dari dalam VS Code.
> Cocok buat kamu yang sering fokus ngoding sampai lupa waktu.

## Cara Kerja
Extension ini membantu mengingatkan waktu sholat dengan menampilkan notifikasi di VS Code.

Alur umumnya:
- Mengambil/menentukan jadwal sholat.
- Menjalankan pengecekan waktu secara berkala.
- Saat memasuki waktu sholat, extension menampilkan notifikasi.

## Fitur
- [x] Pengingat waktu sholat melalui notifikasi di VS Code
- [x] Berjalan otomatis di background saat VS Code aktif

Ringkasan:

| Area | Detail |
| --- | --- |
| Output | Notifikasi di VS Code |
| Mode | Background saat VS Code berjalan |
| Tujuan | Membantu kamu lebih tepat waktu |

## Instalasi (VSIX)
> Catatan: instalasi via VSIX cocok kalau kamu install manual dari file rilis.

1. Download file `.vsix` dari halaman rilis/repository.
2. Buka VS Code.
3. Masuk ke Extensions:
   - Klik ikon Extensions di sidebar, atau
   - Tekan `Ctrl+Shift+X`
4. Klik ikon `...` (More Actions).
5. Pilih `Install from VSIX...`.
6. Pilih file `.vsix` yang sudah kamu download.
7. Reload/Restart VS Code jika diminta.
